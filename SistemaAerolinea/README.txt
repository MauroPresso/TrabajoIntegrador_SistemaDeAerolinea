Integrantes:
- Agustín Almirón
- Juan Cruz Barria
- Mauro Presso

Descripción:
Sistema desarrollado en Java para gestionar una aerolínea aplicando Programación Orientada a Objetos.

El sistema permite:
- Registrar vuelos nacionales, internacionales y charter.
- Registrar pasajeros.
- Gestionar tripulantes.
- Reservar vuelos.
- Cancelar reservas.
- Mostrar vuelos programados.
- Mostrar vuelos ordenados por destino.
- Calcular el total de asientos ocupados.
- Persistir la información entre ejecuciones mediante serialización.

Instrucciones de ejecución:
1. Abrir el proyecto en Visual Studio Code, IntelliJ IDEA, Eclipse o NetBeans.
2. Verificar que el JDK esté instalado correctamente.
3. Compilar el proyecto.
4. Ejecutar la clase Main.
5. Utilizar el menú por consola.
