\doxysubsection{Detailed Description}
Repositorio encargado de guardar y recuperar vuelos desde un archivo binario. 

Esta clase implementa la interfaz \doxylink{interfaceinterfaces_1_1_i_repositorio}{IRepositorio$<$\+Vuelo$>$} y encapsula la lógica de persistencia usando Object\+Output\+Stream y Object\+Input\+Stream.

El archivo utilizado por defecto es data/vuelos.\+dat.
