\doxysubsection{Detailed Description}
Clase abstracta base para todos los tipos de vuelo. 

Implementa \doxylink{interfaceinterfaces_1_1_i_operable}{IOperable} para exponer operaciones comunes como embarcar y cancelar. También implementa Serializable para permitir la persistencia en archivo.
