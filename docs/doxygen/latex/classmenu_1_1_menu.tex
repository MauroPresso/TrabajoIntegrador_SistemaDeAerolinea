\doxysubsection{Detailed Description}
Gestiona la interacción del usuario con el sistema mediante consola. 

Esta clase utiliza Scanner para leer datos ingresados por teclado y delega las operaciones principales en la clase \doxylink{classservicio_1_1_aerolinea}{Aerolinea}.

Todos los ingresos por teclado se validan mediante métodos auxiliares para evitar datos vacíos, números negativos, formatos inválidos, fechas incorrectas, opciones fuera de rango o valores incoherentes.

Además, dispara el guardado de vuelos luego de las operaciones que modifican la lista de vuelos o sus reservas.
