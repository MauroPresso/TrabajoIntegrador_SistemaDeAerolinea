\doxysubsection{Detailed Description}
Excepción personalizada para indicar que un vuelo no puede ser reservado. 

Esta excepción se lanza cuando se intenta reservar un vuelo que se encuentra en vuelo, cancelado o sin asientos disponibles.

Al extender de Exception, se trata de una excepción checked, por lo que debe ser declarada con throws o manejada con try-\/catch.
