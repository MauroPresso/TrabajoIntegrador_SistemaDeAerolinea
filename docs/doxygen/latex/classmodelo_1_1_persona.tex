\doxysubsection{Detailed Description}
Clase base abstracta para representar personas dentro del sistema. 

La clase implementa Comparable para permitir ordenar personas por apellido. También redefine \doxylink{classmodelo_1_1_persona_a92aa979dd693c3554289995958858f2d}{equals()} y \doxylink{classmodelo_1_1_persona_a90955afde4056d0a90820ab7b2d38cad}{hash\+Code()} tomando como identidad única el DNI.
