Detailed Description

Repositorio encargado de guardar y recuperar vuelos desde un archivo binario.

Esta clase implementa la interfaz IRepositorio<Vuelo> y encapsula la lógica de persistencia usando ObjectOutputStream y ObjectInputStream.

El archivo utilizado por defecto es data/vuelos.dat.
